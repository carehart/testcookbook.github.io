Lettuce + Selenium

Lettuce and Selenium

Lettuce Selenium Example

Python 2.7

Now that you have a basic startup of Lettuce, you can add some things so that you can use selenium to test your websites and applications. First a little bit of setup will be needed.

Install packages

requirements.txt

lettuce
selenium
python-Levenshtein

Terminal

pip install -r requirements.txt

There is one more important thing that we will need, geckodriver. Geckodriver is used by Selenium to talk with Firefox. You can download the appropriate one for your machine here https://github.com/mozilla/geckodriver/releases. Once it is downloaded make sure to unpackage it place the file in your PATH. If it is not in the PATH you will get an error.

The Feature

For this experiment we will do a simple test going to google.com and making sure there is Google in the title.

Feature: Google Home Page
  Scenario: Google Title Description
    Given I visit Google
    Then I see title Google

The Steps

from lettuce import *
from selenium import webdriver


@before.each_scenario
def setup_some_scenario(scenario):
    world.driver = webdriver.Firefox()

@after.each_scenario
def setup_some_scenario(scenario):
    world.driver.close()

@step(u'Given I visit Google')
def given_i_visit_google(step):
    world.driver.get("http://www.google.com")

@step(u'Then I see title Google')
def then_i_see_title_google(step):
    assert "Google" in world.driver.title
